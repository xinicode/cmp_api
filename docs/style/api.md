## 自定义样式


### 内置全局样式

| 类名 | 描述 |
| ------ | ------ |
| .i-scrollbar-hide | 滚动条为极简样式 | 
| .i-scrollbar | 滚动条为极简样式 | 
| .i-table-no-border | Table 将去除左右边框 |
